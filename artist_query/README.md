# Music Database

A command line application which reads a CSV file of songs and allows a
user to query it  with the following interface:

```
Query: Taylor Swift

*************************
* Songs by Taylor Swift *
*************************
Back To December
I Knew You Were Trouble
Love Story
We Are Never Getting Back Together

Query: Kanye West

***********************
* Songs by Kanye West *
***********************
All Of The Lights
Dark Fantasy
Gold Digger
Gone
Good Life
Stronger
```
